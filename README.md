# Documentación del Proyecto Infinity Engine para FNF

## Instalación
1. Asegúrate de tener **Node.js** instalado en tu máquina. Puedes descargarlo desde [aquí](https://nodejs.org/).
2. Clona el repositorio:
   ```bash
   git clone https://github.com/ZER0CHEAT/infinity-engine-fnf.git
   ```
3. Navega a la carpeta del proyecto:
   ```bash
   cd infinity-engine-fnf
   ```
4. Instala las dependencias:
   ```bash
   npm install
   ```

## Características
- Motor de juego optimizado para **Friday Night Funkin'**.
- Personalización fácil de personajes y escenarios.
- Soporte para múltiples modos de juego.
- Documentación completa para desarrolladores.

## Instrucciones de Uso
1. Para iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
2. Abre tu navegador y ve a `http://localhost:3000` para jugar.
3. Para compilar el proyecto para producción:
   ```bash
   npm run build
   ```
